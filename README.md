# Stulchik PWA Archive - НЕ РАБОТАЕТ

Цифровой архив текстовых статей с системой индексации и поиска. 
Проект разработан для быстрого доступа к базе из 8000+ материалов.

## Технологический стек
- **Frontend:** PWA (HTML5, CSS3, JavaScript)
- **Database:** SQLite + [sql.js](https://sql.js.org/)
- **Table Engine:** DataTables.net
- **Backend/Automation:** Python (скрипт конвертации)

## Основные возможности
- [x] Автоматическая конвертация .txt (ANSI/LF) в SQLite.
- [x] Обработка множественных категорий для одной статьи.
- [x] Полнотекстовый поиск по архиву.
- [x] Сортировка по категориям, авторам и статусу.
- [x] Адаптивный интерфейс для мобильных устройств (PWA).

## Установка и запуск
1. Клонируйте репозиторий.
2. Поместите исходные .txt файлы в папку согласно скрипту `convert_to_db.py`.
3. Запустите скрипт конвертации для обновления базы `archive.db`.
4. Для публикации используйте GitHub Pages.
